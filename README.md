# Media Pembelajaran Sel Elektrolisis

Aplikasi web interaktif untuk mempelajari konsep sel elektrolisis dengan simulasi dan kuis.

## Fitur

### 1. **Tab Teori**
- Penjelasan lengkap tentang sel elektrolisis
- Komponen-komponen sel elektrolisis
- Hukum Faraday dan rumus perhitungan
- Aplikasi elektrolisis dalam kehidupan sehari-hari

### 2. **Tab Simulasi**
- Simulasi visual proses elektrolisis

- Kontrol tegangan (1-12V)

- Kontrol waktu (1-60 menit)
- Animasi pergerakan ion menuju elektroda
- Perhitungan otomatis massa dan volume produk
- Tampilan reaksi di katoda dan anoda

### 3. **Tab Kuis**
- 5 pertanyaan pilihan ganda
- Feedback langsung untuk setiap jawaban
- Sistem penilaian otomatis
- Navigasi antar pertanyaan

## Cara Menggunakan

1. Buka file `index.html` di browser web modern (Chrome, Firefox, Edge, Safari)
2. Navigasi antar tab menggunakan menu di bagian atas
3. Di tab simulasi:
   - Pilih jenis elektrolit
   - Atur tegangan, arus, dan waktu
   - Klik "Mulai Elektrolisis" untuk memulai animasi
   - Lihat hasil perhitungan di bagian bawah
4. Di tab kuis:
   - Pilih jawaban untuk setiap pertanyaan
   - Klik "Submit Jawaban" untuk melihat hasil
   - Gunakan tombol navigasi untuk berpindah antar pertanyaan
